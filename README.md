# univfyi-embed

Embed UnivFYI widgets on any website. Part of the [FYIPedia](https://fyipedia.com) developer tools ecosystem.

**Coming soon.** Visit [UnivFYI](https://univfyi.com) for more information.

## Embed Widget

Embed [UnivFYI](https://univfyi.com) widgets on any website with [univfyi-embed](https://widget.univfyi.com):

```html
<script src="https://cdn.jsdelivr.net/npm/univfyi-embed@1/dist/embed.min.js"></script>
<div data-univfyi="entity" data-slug="example"></div>
```

Zero dependencies · Shadow DOM · 4 themes (light/dark/sepia/auto) · [Widget docs](https://widget.univfyi.com)
